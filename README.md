## Running examples

Example code is defined in separate, runnable files:

```
demos/src/bin
├── explicit_lifetime.rs
├── lifetime1.rs
├── lifetime2.rs
├── mutability1.rs
├── mutability2.rs
├── mutability3.rs
├── mutable_ref_aliasing.rs
├── result1.rs
├── result2.rs
├── result3.rs
└── type_inference.rs
```

To run an example, e.g.:

```
cargo run --bin type_inference
```
